# プロジェクト名：弓道射法八節姿勢解析アプリケーション開発

## １．目的  
　　弓道の射形動画の姿勢解析から、射法八節の動作を解析するAI（GRU）モデルを作成する。  
  
## ２．機能概要  
### １）動画ファイルを再生する。  
  　　　再生中、キー押下によって以下の操作ができる\
　　　　・再生の一時停止／再開、終了\
　　　　・再生速度の変更（スロー再生、早送り）\
　　　　・画像（フレーム）のスキップ、巻き戻し\
　　　　・繰り返し再生\
　　　　・スクリーンへのグリッド表示、画像の部分拡大（ズームアップ。ただし画素は粗くなる）
### ２）動画ファイルをクリッピングして、別ファイルに保存する。  
　　　　・別ファイル保存したい矩形領域を、マウスでドラッグして指定（平面的なクリッピング）\
　　　　・2番目以降の矩形領域でモザイク表示を指定\
　　　　・別ファイルへの書き出し開始／停止をキー操作で指定（時間的なクリッピング）
### ３）骨格の動きをAI(YOLOモデル)で姿勢解析し、プログラムによるロジック解析および、下記4.のGRUモデルで射法八節の動作状況を判定して表示する。  
　　　　・動画ファイルの射形を解析して、骨格の動き、射法八節の動作状況を表示する\
　　　　・解析結果のデータをファイルに保存、DBへの登録、解析データのプロット表示\
　　　　・解析中の動画を別ファイルに保存
### ４）射形解析モデルのAI学習・検証を実施する。  
　　　　・3)の解析結果データを入力データとして学習（ディープラーニング）し、射形解析モデル(GRU)を作成\
　　　　・3)の解析結果データを使って、学習済射形解析モデルを検証

## ３．開発、動作環境
　　　・PC  :Windows11公式対応PC（推奨：CPU第10世代以降、主メモリ16GB）    
　　　・OS  ：Windows 11  
　　　・操作環境：PowerShellコンソール  
　　　・開発言語：Python 3.12  
　　　・使用AIパッケージ：ultralytics==8.4.17, torch==2.7.0
